# Taproom Chatbot

An embeddable chatbot widget for brewery websites. Answers common taproom questions automatically.

## What It Does
- Embeddable chat widget for any brewery website
- Answers FAQs: hours, location, dog policy, what's on tap, events
- Customizable responses per brewery
- Powered by OpenClaw agent

## Tech Stack
- Simple HTML/JS widget (embeddable)
- OpenClaw agent for natural responses
- Configurable JSON for brewery-specific info
